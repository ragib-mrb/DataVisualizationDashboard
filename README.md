# Data Visualization Dashboard

**System Requirements**

 - Node Version: 20+ 
 - NPM Version: 10+

**Setup Guide**
 - Clone Repo: `git clone https://github.com/ragib-mrb/DataVisualizationDashboard.git`
 - `npm i`
 - `npm run dev`
